# COS782-Assignment-2
A generic c++ implementation of the strategy design pattern

## How to run

In the root folder of the project, run the following command:

### Linux

```bash
./Debug/main
```

### Windows

```bash
Debug/main.exe
```

## How to compile

In the root folder of the project, run the following command:

### Linux

```bash
make clean
```

then 

```bash
make all
```

### Windows

```bash
mingw32-make clean
```

then 

```bash
mingw32-make all
```
